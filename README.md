
# Notes App

A simple yet functional Notes App built with HTML, CSS, and JavaScript.
It allows users to add, view, and delete notes. All notes are stored in the browserâ€™s localStorage, so your data stays even after refreshing the page.

## Features 

- Add new notes with a simple input form
- Display notes as a styled list
- Delete notes individually
- Persistent data storage using localStorage
- Responsive and clean UI

## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- localStorage API

## How to Use
- Clone or download this repository
- Open index.html in your browser
- Type a note and click "Add Note"
- See it added to the list and saved to localStorage
- You can remove any note by clicking the delete button
